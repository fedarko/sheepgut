==========
To figure out the TOTAL NUMBER OF READS in a fastq.gz file:
(assumes that the file has 4 lines per read, which should be the case for
most such files.)
1. Run "zcat reads.fastq.gz | wc -l" -- total number of lines in the file
2. Divide that result by 4

Based on https://www.biostars.org/p/139006/#298653.

==========
To figure out the TOTAL READ LENGTH in a fastq.gz file:

zcat reads.fastq.gz | paste - - - - | cut -f 2 | tr -d '\n' | wc -c 

From https://www.biostars.org/p/78043/#130586.
